# Frontend

Vue 3 + Vite frontend scaffold with typed API integration.

## Quick Start

```bash
npm install
npm run dev
npm run check
```

## Current Stack

- Vue 3
- Vite
- TypeScript

This scaffold currently provides:

- Typed API contracts in `src/types/api.ts`
- Generic HTTP client in `src/api/client.ts`
- Feature-oriented endpoint helpers in `src/api/endpoints.ts`
- Vite entry in `src/main.ts`
- Root app shell in `src/App.vue`
